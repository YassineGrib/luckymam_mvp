## 🎯 Description du Projet

**Luckymam** est une application mobile premium destinée aux **mamans algériennes**, offrant un compagnon de vie complet depuis la **grossesse jusqu'à l'enfance**. L'application permet de :

- 📸 Capturer des moments émotionnels sous forme de "Capsules" (photo + audio)
- 📅 Suivre les jalons importants via une timeline intelligente
- 💉 Gérer le calendrier vaccinal selon le programme national algérien
- 👶 Créer des profils pour chaque enfant
- 🌍 Utiliser l'app en **Français** et **Arabe (RTL)**
